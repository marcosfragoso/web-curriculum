Alguns projetos

HEAC

Desenvolvido com o intuito de otimizar o gerenciamento de arquivos em empresas e prefeituras, este sistema visa facilitar tanto a gestão atual quanto as futuras demandas. A base do sistema foi construída utilizando Django para o back-end, oferecendo robustez e flexibilidade na manipulação de dados, enquanto o front-end foi implementado com Bootstrap, proporcionando uma interface amigável e responsiva. Além disso, o deploy foi realizado na Google Cloud Platform, garantindo escalabilidade e segurança para os dados armazenados.

SUAP do Sertão

Este projeto foi concebido como uma ferramenta de estudo para explorar e compreender o funcionamento de um sistema de gestão acadêmica inspirado no SUAP. Utilizando tecnologias como Java, Spring Boot, Spring Security, PostgreSQL, Thymeleaf, HTML, CSS e Bootstrap, o sistema oferece funcionalidades de autenticação e controle de acesso. A implementação visa proporcionar uma experiência prática no desenvolvimento de aplicações web completas, com foco na segurança e integração com banco de dados. A interface foi projetada com Thymeleaf e Bootstrap, criando um ambiente de aprendizado interativo e visualmente atraente.

API Bancária

Este projeto foi cuidadosamente elaborado como uma ferramenta de estudo para a compreensão prática de APIs bancárias. Ao empregar as tecnologias Java e Spring Boot, o sistema oferece uma abordagem hands-on para explorar e compreender os conceitos essenciais subjacentes ao desenvolvimento de APIs para instituições financeiras. Esta iniciativa não apenas permite a exploração ativa dos fundamentos do desenvolvimento de APIs, mas também oferece uma oportunidade valiosa para aplicar esses conceitos em um contexto realista e prático.

BarberSole App

Este projeto foi concebido como uma ferramenta de estudo para o desenvolvimento de um sistema de gestão para barbearias. Utilizando tecnologias como Java, Spring Boot, Spring Security, PostgreSQL, Thymeleaf, HTML, CSS, Bootstrap, Ajax e Highcharts, o sistema permite o gerenciamento de clientes, serviços e agendamentos. A interface foi desenvolvida com Thymeleaf e Bootstrap, proporcionando uma experiência de usuário amigável e responsiva. Com o uso de Ajax, o sistema oferece interatividade em tempo real, e os gráficos gerados pelo Highcharts fornecem visualizações detalhadas de dados, auxiliando na tomada de decisões.
